# STRIKE ARENA 🔫

Мультиплеерный 3D FPS-шутер в стиле Counter-Strike. Three.js + Node.js + Socket.IO.

## Запуск локально

```bash
npm install
ADMIN_PASS=secret npm start
# открой http://localhost:3000
```

## Деплой (Railway / любой Node-хостинг)

Просто задеплой репозиторий — сервер слушает `process.env.PORT` (по умолчанию 3000). Задай переменную окружения `ADMIN_PASS` для доступа к админ-панели (`/admin`). Игроки заходят по одной ссылке и попадают на общий сервер.

## Фичи v2

- 🗺 **3 карты**: de_arena (трёхлайновая), de_dust_mini (классика, два плэнта), aim_yard (тренировочная) — смена карты из админки
- 🔫 **8 оружий**: нож, USP-9, Desert Eagle, MP5-SD, Nova (дробовик), AK-47, M4A4, AWP (скоуп, ваншот)
- 💰 **Раунды и экономика как в CS**: фриз-тайм, таймер раунда, награды за победу/убийства, магазин закупки (B), матч до 10 побед
- 🛡 **Защита спавна** — 3 секунды неуязвимости после возрождения
- 🎨 **Скины оружия** — 7 расцветок, выбираются в лобби
- ✋ **Руки от первого лица** — видно, как держишь оружие
- 📱 **Мобильное управление** — джойстик, кнопки огня/прыжка/перезарядки, можно играть с телефона
- 🛠 **Админ-панель** `/admin` — игроки, кик/бан/мут, смена карты, рестарт раунда/матча, объявления, логи
- 🛡 **Серверная защита от читов**: урон/патроны/скорострельность/закупка считаются на сервере, проверка прямой видимости, анти-телепорт со снапбэком, авто-кик читеров
- 🚦 **Рейт-лимиты**: спам пакетами/чатом/коннектами режется на сервере
- 💥 Отдача, разброс при движении, перезарядка, хедшоты, киллфид, таблица счёта (Tab), чат (Enter), миникарта
- 🧍 Анимированные 3D-персонажи двух команд, 🔊 процедурный звук (выстрелы, таймер, старт/конец раунда)

## Управление

| Клавиша | Действие |
|---|---|
| WASD | движение |
| Shift | тихий шаг |
| Ctrl / C | присед |
| Space | прыжок |
| ЛКМ | огонь |
| ПКМ | скоуп (AWP) |
| R | перезарядка |
| 1-4 | смена оружия |
| B | магазин (в закупку) |
| Tab | счёт |
| Enter | чат |
